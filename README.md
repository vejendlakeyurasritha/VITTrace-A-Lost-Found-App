# VITTrace – Lost & Found App

VITTrace is an Android application designed to help students report, track, and recover lost items within the university campus. The app allows users to post lost or found items and search through reported items easily, improving the chances of recovering belongings.

## 📱 Features

* Report lost items with details
* Post found items
* Search and browse reported items
* Simple and user-friendly interface
* Helps students reconnect with their lost belongings

## 🛠 Tech Stack

* Android Studio
* Kotlin
* Gradle (Kotlin DSL)
* XML (UI Design)

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/yourusername/VITTrace.git
```

2. Open the project in **Android Studio**

3. Sync Gradle dependencies

4. Run the application on:

   * Android Emulator
   * Physical Android device

## 🚀 How It Works

1. Users can report an item as **Lost** or **Found**.
2. The information is stored and displayed in the app.
3. Other users can browse the list and identify matching items.
4. This helps students recover their lost belongings efficiently.
